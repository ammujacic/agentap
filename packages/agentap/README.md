# agentap

Mobile bridge for AI coding agents. Control your AI coding assistants from anywhere.

## Quick Start

```bash
npx agentap
```

Or install globally:

```bash
npm install -g agentap
agentap
```

## What is Agentap?

Agentap connects your AI coding agents (like Claude Code) to your mobile device, letting you:

- Monitor agent activity in real-time
- Approve or reject agent actions
- Receive push notifications for important decisions
- Control your agents from anywhere
